The School Attendance System is a browser-based attendance management tool designed for schools and colleges. It works entirely offline after download, requires no installation, and stores all data locally on the user's device.

The system is organized into three levels: Classes, Months, and Subjects. Each class or section can have its own class teacher, multiple months, and multiple subjects, with each subject assigned to a different teacher. Attendance is maintained separately for every subject, allowing users to switch between subjects easily.

For each subject, users can create session dates and mark student attendance as Present, Absent, or Unmarked with a single click. A live summary automatically displays each student’s present count, absent count, and attendance percentage. Students are entered only once per month and are shared across all subjects, eliminating repetitive data entry.
